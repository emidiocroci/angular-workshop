Direttive


Istruzioni che indicano ad angular che alcuni elementi del DOM hanno funzionalità aggiuntive


Possono essere


tag / attributi / classi / commenti


vanno usate (create) tutte le volte in cui si ha la necessità di manipolare elementi del DOM o di interagire con la view


non usare il CONTROLLER


ok, tutto molto bello... ma come si creano?











restrict: tag (E) / attr (A) / comment (M) / class (C)

template: html che verrà (se necessario) inserito nel DOM

scope: stringa (@) / binding (=) / funzione (&)

link: utilizzata per registrare eventi del DOM e setup del binding

controller: utilizzata per comunicazione tra direttive
